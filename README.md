# SKFE

![SKPM](https://img.shields.io/badge/skpm-great-ff69b4.svg)

⭐️ SKFE is a Sketch plugin designed to save life of front-end engineer. Still in development stage~

### HOW TO USE
- Install plugin
- `Command` + `Alt` + `P`
- Select the layer
- Get the scss code ~:tada::tada::tada:
